# Contributing

Thank you for considering a contribution to HeronWin.

By submitting a pull request, patch, issue comment with code, documentation
change, or other contribution, you agree that your contribution is your original
work or that you have permission to submit it. Unless you clearly state
otherwise, contributions are licensed under the Apache License, Version 2.0, the
same license that covers this repository.

Please do not submit secrets, credentials, private account data, personal
profile information, generated evidence bundles, screenshots containing private
content, or third-party content that you do not have permission to publish.

Before opening a pull request, run the relevant tests where practical:

```powershell
dotnet test src\heronwin.sln
```
