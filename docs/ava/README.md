# AVA Documentation

AVA is the HeronWin accessibility validation assistant. It runs scenario-backed validation against Windows apps and browser workflows, collects UI evidence, runs deterministic accessibility checks, and writes Markdown/JSON reports.

Use AVA when the question is not only "can HeronWin do this workflow?" but also "what accessibility evidence and findings did the workflow produce?"
